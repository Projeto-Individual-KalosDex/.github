🟦 KalosDex Organization

Bem-vindo à organização oficial do projeto **KalosDex**.

Projeto acadêmico individual voltado ao desenvolvimento de uma plataforma web inspirada no universo Pokémon, com foco na região de Kalos.

🎯 Objetivos

- Aplicar conhecimentos em HTML, CSS e JavaScript
- Utilizar banco de dados relacional com SQL
- Desenvolver autenticação de usuários
- Criar dashboards interativos

📁 Repositórios

- `SiteHTML-CSS`
- `ScriptSQL`
- `Documentacao`

🛠 Ferramentas

- HTML
- CSS
- JavaScript
- MySQL
- GitHub

🌟 Tema

Inspirado na franquia Pokémon e na região de Kalos.
